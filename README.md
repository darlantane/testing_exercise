# Software Quality Assurance Exercise Project

This repository showcases a comprehensive project completed as part of a software testing training. It puts into practice the fundamental skills acquired, from product analysis to test execution and automation.

## Technologies Used

*   **Test Management** : Documented methodology and tools like Squash TM.
*   **Automation** : Scripts written in **Python**.
*   **Project Management** : Agile approach and User Story writing.

## Learning Objectives

*   Apply a complete testing process, from analysis to reporting.
*   Learn to produce professional test documentation (strategy, plans, test cases, reports).
*   Get started with test automation using Python.
*   Work within a quality-driven Agile framework.
